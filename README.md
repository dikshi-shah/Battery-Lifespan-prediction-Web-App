# Battery Lifespan Prediction Web App

A deep learning model predicts battery **Remaining Useful Life (RUL)** by analyzing real usage patterns: charging time, voltage drops, discharge time, and related metrics. The project includes a Streamlit web app with a **dashboard** for exploring the dataset and a **prediction page** where users can input battery parameters and get an instant RUL estimate in cycles.

## Features

- **Dashboard**: Filter by cycle index range, view charging/discharge trends, voltage metrics, and time-at-voltage charts.
- **Prediction**: Enter cycle and sensor values to get a predicted remaining useful life (cycles).
- **Tech stack**: TensorFlow/Keras (model), scikit-learn (preprocessing, PCA), Plotly (charts), Streamlit (UI).

## Requirements

- Python 3.10 (see `Streamlit/runtime.txt` for deployment)
- Dependencies in `Streamlit/requirements.txt`

## Setup and run

1. **Clone the repository** and open a terminal in the project root (the folder that contains `Streamlit/` and `Battery_RUL.csv`).

2. **Create a virtual environment** (recommended):
   ```bash
   python -m venv venv
   venv\Scripts\activate   # Windows
   # source venv/bin/activate   # macOS/Linux
   ```

3. **Install dependencies**:
   ```bash
   pip install -r Streamlit/requirements.txt
   ```

4. **Model artifacts**: The app expects `Model.pkl`, `scaler.pkl`, and `PCA.pkl` in the `Streamlit/` folder. If they are not included in the repo, train the model using the notebook `battery_life_DL.ipynb`, then copy the generated `.pkl` files into `Streamlit/`.

5. **Run the app** (from the **project root**):
   ```bash
   streamlit run Streamlit/main_page.py
   ```
   Then open the URL shown in the terminal (e.g. http://localhost:8501).
